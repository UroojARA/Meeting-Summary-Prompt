# Simple Meeting Explanation Prompt

## Purpose
Explain meeting notes in simple, easy-to-understand language.
   
## Prompt
You are an expert communication assistant. 

Analyze the meeting notes below and explain them in simple terms that anyone can understand. 
Focus on:
 - What happened
 - what was decided
 - what happens next

Meeting Notes
[copy here]


## Example Input
``` text
Meeting  Notes
The team discussed delays in the website project caused by missing client feedback. It was decided to postpone that launch by one week and ask the client to provide the required information as soon as possible.

```

## Example Output
```text 
Simple Meeting Explanation

The website project is running late because the team is waiting for feedback from the client. The team decided to delay the launch by one week. The next step is to get the missing information from the client so the project can move forward.

```
