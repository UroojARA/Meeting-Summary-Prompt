# Team Standup Summary Prompt

## Purpose
Summarize daily standup meetings by capturing completed work, current tasks, and blockers.

## Prompt
You are an expert project cordinator.

Analyze the team standup notes below and create a concise summary.

Include the following sections:
- Completed Tasks
- Current Work
- Blockers
- Next step

Standup Notes:
{copy here}

## Example Input

```text
Team Standup Notes

 - John compeleted the login page.
 - Maria is working on the payment module.
 - David is blocked because he is waiting for API access.
```

## Example Output

``` text
Team Staandup Summary

Completed Tasks:
- John completed the login page.

Current Work:
- Maria is working on the payment module.

Blockers:
 - David is waiting for API access.

Next Steps:
 - Provide API access to David.
- Continue development of the payment module.
```
