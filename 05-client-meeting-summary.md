# Client Meeting Summary Prompt

## Purpose

You are an expert business assistant.

Analyze the client meeting notes below and create a professional summary.

Include the following sections:

 - Meeting Overview
 - Client Requests
 - Decisions Made
 - Action Items
 - Next Steps

Meeting Notes:
{Copied}

## Example Input
``` text
Client Meeting Notes

- The client requested a new dashboard feature.
- Sarah will send the project proposal by Friday.
- The next meeting is scheduled for next Tuesday.
 ```

## Example Output
```text
Client Meeting Summary

Meeting Overview:
Discussion about new feature requirements and project planning.

Client Requests:
 - New dashboard feature.

Action Items:
Sarah to send the project proposal by Friday.

Next Steps:
 - Followup meeting scheduled foe next Tuesday. 

```
